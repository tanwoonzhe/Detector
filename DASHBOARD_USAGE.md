## 🎯 如何使用

### 1. 启动实时 Dashboard
```powershell
streamlit run app/dashboard_realtime_binance.py
```

### 2. 在侧边栏配置
- ✅ **勾选 "🤖 启用 AI 预测"**
- 选择模型：GRU 或 LightGBM
- 设置刷新间隔：5-60秒
- 选择 K 线周期：1分钟到4小时
- 调整历史数据天数：1-30天

### 3. 查看预测结果
- 📈 **看涨** - 绿色，建议买入或持有
- 📉 **看跌** - 红色，建议卖出或观望  
- ➡️ **震荡** - 黄色，建议保持观望
- 📊 **概率分布图** - 显示各方向的置信度

## ⚙️ 配置选项

### 刷新间隔
- **5秒** - 超快速（适合短线交易）
- **10秒** - 快速
- **15秒** - 标准（推荐）
- **30秒** - 较慢
- **60秒** - 慢速

### K线周期
- **1分钟** - 超短线
- **5分钟** - 短线（推荐预测）
- **15分钟** - 中短线
- **1小时** - 中线
- **4小时** - 长线

### 历史数据
- **1-7天** - 适合短期分析
- **7-14天** - 适合中期分析
- **14-30天** - 适合长期分析

## 🔧 故障排除

### 问题: 预测功能不可用
**原因**: 模型还未训练
**解决**: 
```powershell
python train.py --model gru --epochs 100 --batch-size 64
```

### 问题: Dashboard 显示错误
**原因**: 网络连接问题
**解决**: 
1. 检查网络连接
2. 确认可以访问 Binance API
3. 点击"立即刷新"按钮

### 问题: 预测显示"数据不足"
**原因**: 历史数据少于24小时
**解决**: 增加历史数据天数到至少2天

### 问题: 加载很慢
**原因**: 特征工程计算量大
**解决**: 
1. 减少历史数据天数
2. 增加刷新间隔
3. 暂时关闭预测功能
